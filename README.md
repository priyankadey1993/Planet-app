# Planet-app
Planet Information App is a Qt Widgets and C++ desktop application that displays information about planets in a clean and interactive way. Each planet is shown in a separate tab, with its image displayed on the tab page. When the user clicks the Details button, a dialog opens showing more information about that planet. The project uses QTabWidget, QWidget, QDialog, signals and slots, and a custom data structure to keep the code modular and easy to extend.
